## Description

React Typescript Template with React, Typescript, ESLint, TravisCI, Commitizen, Yarn

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# build
$ yarn build

# development
$ yarn start

# watch mode
$ yarn start
```

## Test

```bash
# unit tests
$ yarn test

# lint
$ yarn lint
```
